# Instalation

- npm install
- npm run dev

# Backend Firebase
```
Timer/src/firebase.js
```
Create a file .env for VITEJS in your root folder and fill `#`

``` 
  VITE_API_KEY="#",
  VITE_AUTH_DOMAIN="#",
  VITE_DATABASE_URL="#",
  VITE_PROJECT_ID="#",
  VITE_STORAGE_BUCKET="#",
  VITE_MESSAGE_SENDER_ID="#",
  VITE_APP_ID="#",
  VITE_MEASUREMENT_ID="#"
```
